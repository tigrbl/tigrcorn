# Notes index

This folder is the mutable note/work-tracking area.

Files:

- `inprog.md` - what remains, what is frozen, and what is explicitly out of scope for the current line
- `feat_matrix.md` - master feature target matrix for current targets and future target waves
- `feature_reg.md` - working register of current supported targets and candidate feature waves
- `risk_reg.md` - mutable working risk register derived from issues, feature targets, and claim tiers
- `issue_reg.md` - researched GitHub issue register with per-issue summary and disposition
- `issue_mat.md` - issue register matrix for sorting by state, subsystem, risk, and next action
