# Intermediary / proxy corpus

This seed corpus collects the currently preserved HTTP/1.1 third-party artifact plus carrier-specific CONNECT relay vector metadata.
It improves repository transparency for intermediary/proxy-style behavior, but it does not replace broader third-party proxy interoperability evidence.

This historical seed corpus remains preserved for provenance. The newer minimum certified intermediary / proxy-adjacent corpus lives in `docs/review/conformance/intermediary_proxy_corpus_minimum_certified/`.
