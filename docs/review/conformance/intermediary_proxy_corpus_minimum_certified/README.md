# Minimum certified intermediary / proxy corpus

This corpus promotes one preserved third-party forwarding artifact per applicable carrier into a minimum certified intermediary/proxy-adjacent evidence root.

Included minimum certified cases:

- HTTP/1.1 origin-form / request-forwarding via curl
- HTTP/2 forwarding semantics via h2
- HTTP/3 forwarding semantics via aioquic

Supplemental local-vector cases remain in-tree for CONNECT, trailers, and content-coding semantics until broader third-party proxy captures are preserved.
