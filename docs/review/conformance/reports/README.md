# RFC and conformance reports

This directory contains RFC status reports, hardening reports, and related review documents that used to accumulate at the repository root.

Use these locations:

- RFC certification status: `docs/review/conformance/reports/RFC_CERTIFICATION_STATUS.md`
- RFC hardening report: `docs/review/conformance/reports/RFC_HARDENING_REPORT.md`
- additional RFC and reconciliation reports: this directory

Root-level `RFC*.md` reports are intentionally not used; they were consolidated here to keep the repository root focused on package entrypoint documents.
