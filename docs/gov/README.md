- `mut.md` — mutable/immutable folder rules and flip procedure
- `code.md` — code style, code behavior, and public-surface discipline
- `commit.md` — commit message and change-isolation rules
- `pr.md` — pull-request review and validation rules
- `release.md` — versioning, certification, promotion, and release-close workflow
